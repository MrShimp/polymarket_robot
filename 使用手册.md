# 如何使用
## step 1
pip install -r requirements.txt

## step 2
切换网络到香港节点

## step3
test.py中key用你们自己的钱包的eth地址
执行 pip test.py
控制台输出：ApiCreds(api_key='********', api_secret='**********', api_passphrase='4*****') 
把这些都配置到config/sys_config.py

## step4
执行从现在开始每15分钟的市场
python btc_auto_trader.py 1 
1可以替换成任意你想要一次性交易的金额

## step5
控制台查看策略执行日志
tail -f data/btc_strategy_logs/btc_15min_*.log
